# カタログ課題1「絶対に遅刻しないタイマー」

## 環境
* Ruby 3.0.1
* Rails 6.0.4.8

## 実行手順
1. 任意のディレクトリにgit cloneする
```
$ git clone git@github.com:erikoenuma/chikoku_timer.git
```
2. ディレクトリに移動
```
$ cd chikoku_timer
```
3. 以下のコマンドを実行
```
$ bundle install
$ rails db:create
$ rails db:migrate
```
4. サーバーを起動
```
$ rails s
```
5. 以下のURLにアクセスする
```
$ http://localhost:3000/
```
